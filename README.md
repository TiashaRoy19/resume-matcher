# Resume Matcher & Skill Analyzer

## Overview
This project is a Python-based tool that analyzes how well a candidate's resume matches a given job description.  
It extracts key skills from both documents and calculates a match percentage using basic NLP techniques.

## Features
- Resume parsing from PDF files
- Job description text analysis
- Skill extraction using NLP
- Match percentage calculation
- Identification of missing skills

## Tech Stack
- Python
- Natural Language Processing (NLP)
- Regular Expressions (Regex)
- PDF Parsing

## Project Structure

## Use Case
This tool can help job seekers evaluate their resume against job descriptions and understand skill gaps before applying.
